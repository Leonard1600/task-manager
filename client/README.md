# Task Manager - Frontend

Este es el frontend de la aplicación de gestión de tareas, construido con **React** y **Vite**. Utiliza **Tailwind CSS** para el diseño y **Axios** para las peticiones HTTP al backend.

## Requisitos

- **Node.js** (v14.x o superior)

## Instalación y configuración

### 1. Clonar el repositorio:

```bash
git clone https://github.com/tu_usuario/tu_repositorio.git

2. Ir a la carpeta del frontend:
cd frontend

3. Instalar las dependencias:
npm install

4. Iniciar el servidor de desarrollo:
npm run dev


El frontend debería estar corriendo en http://localhost:3000.

Despliegue

Puedes desplegar el frontend en plataformas como Vercel, Netlify o cualquier plataforma que soporte React.

Rutas del Frontend

/login: Página de inicio de sesión

/register: Página de registro de usuario

/dashboard: Página principal donde se muestran las tareas

/tasks/create: Página para crear una nueva tarea

/tasks/edit/:id: Página para editar una tarea existente
