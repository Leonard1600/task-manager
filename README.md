# Task Manager App - Full Stack

Este es un proyecto de gestión de tareas full-stack construido con **React** en el frontend y **Node.js/Express** en el backend. Los usuarios pueden crear, editar, eliminar y ver tareas. El proyecto utiliza **MongoDB** para el almacenamiento de datos y **JWT** para la autenticación.

## Estructura del proyecto

- **Frontend**: [Leer README del Frontend](./frontend/README.md)
- **Backend**: [Leer README del Backend](./backend/README.md)

## Requisitos

- **Node.js** (versión recomendada: v14.x o superior)
- **MongoDB** (usa MongoDB Atlas para el despliegue)
